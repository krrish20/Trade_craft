# **App Name**: Tradecraft Academy

## Core Features:

- Curriculum Delivery: Structured lesson plans covering beginner to advanced trading concepts, with a focus on trading psychology, risk management, and discipline. Lessons are bundled locally for offline access.
- Interactive Quizzes: Interactive quizzes at the end of each lesson to test comprehension, with explanations for each answer to reinforce learning. Boss quizzes at the end of each level gate progress.
- Gamified Learning: Gamified elements such as XP, badges, learning streaks, and level progression to encourage consistent learning and engagement.
- Personalized Study Path: AI-powered personalized study paths based on the user's initial skill assessment and time commitment, guiding them through the curriculum efficiently. The AI tool will estimate study duration based on available lessons and user preferences.
- Risk Management Tools: Interactive tools such as a position size calculator, expectancy visualizer, and daily loss limit planner to help users apply risk management principles.
- Certificate Generation: Downloadable certificate of completion with the user's name, completion date, and total study hours upon finishing the entire curriculum.
- Offline Persistence: Offline persistence of user progress, quiz scores, and settings using LocalStorage, with import/export functionality and a reset option.

## Style Guidelines:

- Primary color: Indigo (#4B0082) to convey a sense of focus and depth.
- Background color: Light gray (#F0F0F0) provides a warm, neutral backdrop for comfortable reading.
- Accent color: Teal (#008080) will highlight important information and calls to action, for excellent contrast.
- Headline font: 'Inter' (sans-serif) for headings and user interface elements, with a modern, readable style.
- Body font: 'Literata' (serif) for lesson content to improve readability and provide a traditional, academic feel.
- Use a consistent set of Lucide icons with a uniform stroke weight, indicating reliability.
- Clean and distraction-free layout with rounded corners and soft shadows to create a premium and focused learning environment.
- Subtle motion effects using Framer Motion for transitions and interactive elements, providing visual feedback and engagement.
